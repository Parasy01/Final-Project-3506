# Calorie Tracker

Calorie Tracker is a Flask-based web application developed for the COSC3506 Software Engineering final project.

## Features
- Add food entries
- Edit food entries
- Delete food entries
- Search food by name or category
- Track daily calorie total
- View calorie chart
- Dark mode support

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap
- Chart.js

## How to Run
1. Open the project folder in VS Code.
2. Create and activate a virtual environment.
3. Install Flask:
   pip install Flask
4. Run the app:
   python app.py
5. Open in browser:
   http://127.0.0.1:5000/
